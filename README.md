# 4X4 数独游戏

一个使用 Visual Basic 开发的 4×4 数独和游戏程序。

## 项目概述

本项目是一个交互式的 4×4 数独游戏应用程序，用户可以选择不同的数独题目进行求解。

## 功能特性

- 🎮 **交互式游戏** - 支持选择不同的数独题目
- 📋 **多个题目** - 包含 16+ 个预设的数独题目
- 📖 **游戏规则说明** - 内置规则提示功能
- 🎯 **4×4 网格** - 需要填充数字 1-4

## 游戏规则

数独游戏的目标是将数字 1-4 填充到 4×4 的方块中，满足以下条件：

- ✓ 每一行中的数字 1-4 各出现一次
- ✓ 每一列中的数字 1-4 各出现一次
- ✓ 每个 2×2 的小方块中的数字 1-4 各出现一次
- ✓ 游戏拥有 16 个可移动的方块

## 项目结构

```
├── 工程1.vbp              # VB 项目文件
├── 苏度库4X4.frm          # 主游戏窗体（游戏界面）
├── 解志飞.frm             # 规则说明窗体（规则提示）
└── 01-7.9元数独题/       # 数独题目库
    ├── a7-001.txt
    ├── a7-002.txt
    ├── a7-003.txt
    └── ... (共16个题目文件)
```

## 使用方法

1. 打开项目文件 `工程1.vbp`
2. 在 Visual Basic IDE 中运行程序
3. 从下拉框选择要解决的数独题目（例如：a7-001）
4. 点击"开始"按钮开始游戏
5. 点击"说明"按钮查看游戏规则
6. 通过移动方块并填入相应的数字，求解数独

## 技术栈

- **开发语言**：Visual Basic 6.0
- **UI 框架**：VB Forms
